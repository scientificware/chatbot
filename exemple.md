### ** Comment commencer la conception d'un cours de sécurité ?** <a name="comment-commencer"></a>
Pour concevoir un cours de sécurité en **Lycée Professionnel**, suivez ces étapes :

1. **Analyser le référentiel** :
   Consultez le référentiel officiel du **Bac Pro Métiers de la Sécurité** sur [Eduscol](https://eduscol.education.fr/).
   - Exemples de thèmes : Prévention des risques, secourisme, gestion des incidents.

2. **Définir les objectifs pédagogiques** :
   Identifiez les **compétences à acquérir** :
   - Prévention des risques professionnels.
   - Gestion des urgences (incendie, accident).
   - Respect des normes de sécurité.

3. **Adapter au public** :
   Prenez en compte le **niveau des élèves** (seconde, première, terminale) et leurs prérequis.

4. **Choisir une progression** :
   Organisez les séquences :
   - **Théorie** → **Pratique** → **Évaluation**.
   - Exemple : Cours sur les EPI → Atelier manipulation d'extincteurs → QCM.

5. **Prévoir des supports** :
   Utilisez des **vidéos** (INRS), des **quiz interactifs** (Kahoot), ou des **ateliers pratiques**.

### **📌 Quelles méthodes pédagogiques utiliser ?** <a name="methodes-pedagogiques"></a>
Pour un cours de sécurité, privilégiez ces **méthodes pédagogiques** :

🎯 **Pédagogie active** :
- **Études de cas** : Analysez un **accident du travail réel** (ex: chute de hauteur, exposition à des produits chimiques).
- **Jeux de rôle** : Simulation d'**évacuation incendie** ou de **premiers secours**.

💻 **Outils numériques** :
- **Vidéos** : Chaîne YouTube de l'[INRS](https://www.youtube.com/user/INRSofficiel).
- **Quiz interactifs** : [Kahoot](https://kahoot.com/), [Socrative](https://www.socrative.com/).
- **Simulateurs** : Logiciels comme **RiskAssess** (évaluation des risques TMS).

🔧 **Travaux pratiques** :
- Ateliers avec du **matériel de sécurité** (extincteurs, EPI).
- Visites d'**entreprises** ou de **centres de formation**.

📝 **Évaluation formative** :
- QCM en ligne (ex: [Google Forms](https://forms.google.com/)).
- Mises en situation filmées et analysées en classe.

### **📌 Comment évaluer les élèves ?** <a name="evaluation"></a>
L'évaluation en sécurité peut inclure :

- **QCM** sur la réglementation (ex: Code du travail, normes NF).

- **Mises en situation** : Utilisation d'un extincteur, premiers secours (PSC1).

📊 **Projets** :
- **Étude de cas** : Élaboration d'un **plan de prévention** pour un atelier.
- **Dossier technique** : Analyse des risques dans un environnement professionnel.

📋 **Grilles d'observation** :
- Évaluez les **comportements** (ex: respect des consignes, port des EPI).

**📌 Ressources** :
- Grilles officielles du **Bac Pro**.

### **📌 Quelles normes de sécurité enseigner ?** <a name="normes"></a>
Les **normes et réglementations** essentielles en sécurité :

📜 **Code du travail** :
- **Articles R4121-1 à R4121-4** : Obligations de l'employeur en matière de sécurité.
- **Article L4121-1** : Principe général de prévention.

🏷️ **Normes NF** :
- **NF EN ISO 45001** : Management de la santé et sécurité au travail.
- **NF X50-100** : Document unique d'évaluation des risques.
- **NF EN 340** : Exigences générales pour les EPI.

⚡ **Règles spécifiques** :
- **EPI** : Normes **EN 388** (gants), **EN ISO 20345** (chaussures de sécurité).
- **Incendie** : Normes **NFPA**, réglementation **ERP** (Établissements Recevant du Public).
- **Risques chimiques** : Réglementation **CLP** (Classification, Labelling, Packaging).
- **Risques électriques** : Norme **NF C 18-510** (travaux électriques).

**💡 Conseil** :
Utilisez des **fiches mémo** pour les élèves avec les normes clés.

### **📌 Quels EPI enseigner ?** <a name="epi"></a>
Les **Équipements de Protection Individuelle (EPI)** à enseigner en Bac Pro Sécurité :

🪖 **Protection de la tête** :
- **Casques** : Norme **EN 397** (protection contre les chocs).

👀 **Protection des yeux et du visage** :
- **Lunettes de sécurité** : Norme **EN 166** (protection contre les projections).
- **Écrans faciaux** : Norme **EN 1731** (protection contre les UV, les étincelles).

👂 **Protection auditive** :
- **Bouchons d'oreille** : Norme **EN 352-2** (pour les bruits < 100 dB).
- **Casque anti-bruit** : Norme **EN 352-1** (pour les bruits > 100 dB).

🤲 **Protection des mains** :
- **Gants** : Norme **EN 388** (résistance mécanique, chimique, thermique).

👟 **Protection des pieds** :
- **Chaussures de sécurité** : Norme **EN ISO 20345** (embout de protection, semelle anti-perforation).

👕 **Protection du corps** :
- **Vêtements haute visibilité** : Norme **EN ISO 20471** (pour les travaux sur route).
- **Combinaisons** : Norme **EN 14605** (protection contre les produits chimiques).

😷 **Protection des voies respiratoires** :
- **Masques FFP** : Norme **EN 149** (FFP1, FFP2, FFP3 selon le niveau de filtration).

**🎯 Activité pratique** :
- Faites **essayer les EPI** aux élèves pour qu'ils comprennent leur importance.
- Organisez un **atelier de reconnaissance des pictogrammes** sur les EPI.

### **📌 Comment organiser une séance pratique ?** <a name="seance-pratique"></a>
Pour organiser une **séance pratique** en sécurité :

1. **Préparer le matériel** :
   - Extincteurs (pour les exercices d'incendie).
   - EPI (casques, gants, chaussures de sécurité).
   - Mannequins (pour les exercices de secourisme).

2. **Définir les objectifs** :
   - Exemple : "Savoir utiliser un extincteur à poudre".
   - "Reconnaître les pictogrammes de sécurité".

3. **Diviser la classe en groupes** :
   - 4-5 élèves par groupe pour une meilleure interaction.

4. **Alterner théorie et pratique** :
   - **10 min de théorie** → **20 min de pratique**.
   - Exemple : Expliquer le fonctionnement d'un extincteur → Faire un exercice d'extinction.

5. **Évaluer les élèves** :
   - Grille d'observation (respect des consignes, manipulation correcte).
   - QCM à la fin de la séance.

**💡 Astuce** :
Filmez les exercices pour une **analyse collective** en classe.

### **📌 Quels logiciels utiliser pour la prévention des risques ?** <a name="logiciels"></a>
Voici des **logiciels utiles** pour la prévention des risques :

🖥️ **Évaluation des risques** :
- **[RiskAssess](https://www.riskassess.fr/)** : Outil d'évaluation des risques TMS (Troubles Musculo-Squelettiques).
- **[EvaRisk](https://www.evarisk.com/)** : Logiciel pour le **Document Unique**.

📊 **Gestion de la sécurité** :
- **[SécuriTech](https://www.securitech.fr/)** : Logiciel de gestion des EPI et des formations.
- **[QHSE Manager](https://www.qhse-manager.com/)** : Outil complet pour la gestion QHSE (Qualité, Hygiène, Sécurité, Environnement).

🎓 **Formation en ligne** :
- **[INRS Formation](https://www.inrs.fr/formation/)** : Modules e-learning sur la prévention.
- **[AFNOR Formation](https://www.afnor.org/formation)** : Formations certifiantes.

**💡 Conseil** :
Utilisez ces outils pour **illustrer vos cours** ou pour des **travaux pratiques** avec les élèves.
